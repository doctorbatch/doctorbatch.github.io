About

Doctor Batch's Magic Sauce is an API for developing batch applications in the Java programming language targetted for deployment to IBM's WebSphere Batch or WebSphere Liberty Profile's Java EE Batch runtime environments. The magic sauce provides portability between the two runtimes by abstracting application development away from the runtime-specific API, and providing adapters for deployment.

The Magic Sauce is being released as an Apache 2.0 License open-source project, and will be available soon from IBM WebSphere Developerworks's GitHub repository.
